# yandex_afisha_project
Яндекс Афиша — это сервис, позволяющий пользователям узнавать о мероприятиях и покупать билеты через партнерские билетные системы.

Маркетинговая команда столкнулась с необходимостью не только привлекать новых пользователей, но и повышать уровень их удержания. Ключевая задача — увеличить долю клиентов, которые возвращаются на платформу после первой покупки.

Цель исследования: провести анализ данных для формирования портрета пользователей с высокой вероятностью повторных заказов.

Результаты анализа позволят:

Быстро выявлять перспективных клиентов и предлагать им персонализированные условия.
Точно настраивать рекламу на аудитории с высокой вероятностью возврата.
Оптимизировать маркетинговые бюджеты.
Повысить общий уровень удержания клиентов.
В рамках проекта будет проведен анализ поведения пользователей, сформированы их профили и выделены ключевые факторы, влияющие на принятие решения о повторной покупке.

Проект состоит из 5 последовательных этапов:

Исследование данных: первичное подключение к SQL-базе и знакомство с данными

Подготовка SQL-запросов для подготовки выгрузки данных

Анализ и исследование данных: выгрузка данных, предобработка и очистка данных, формирование профилей пользователей, выявление закономерностей и распределений и корреляционный анализ

Бизнес-инсайты: формулировка выводов и рекомендаций для бизнеса

Публикация в Git: подготовка и размещение проекта в репозитории GitHub
